Realtime Subtitle Translator (Chrome Extension)
Last updated / 最後更新：2026-07-09

Summary / 摘要： This extension uses your own translation API key (BYOK). Your key and settings stay on your device. Caption text is sent only to the translation provider you choose—not to the extension developer.

本擴充功能使用你自己的翻譯 API key（自備金鑰）。金鑰與設定只存在本機。字幕文字只會送到你選擇的翻譯服務商，不會送到擴充功能作者的伺服器。

EN 1. Who we are

This policy describes how the browser extension Realtime Subtitle Translator (“the Extension”) handles information when you install and use it. The Extension translates on-screen YouTube captions into Traditional Chinese (and optional bilingual display) using an OpenAI-compatible API that you configure.

ZH 1. 我們是誰

本政策說明 Chrome 擴充功能 Realtime Subtitle Translator（「本擴充功能」）在安裝與使用時如何處理資訊。本擴充功能可將 YouTube 畫面上的字幕翻譯成繁體中文（可選雙語顯示），並透過你自行設定的 OpenAI-compatible API 完成翻譯。

EN 2. Data stored on your device

The Extension may store the following in Chrome local storage (chrome.storage.local) on your computer only:

  • Selected translation provider and model name
  • API endpoint URL
  • API key you paste into the setup screen
  • Display preferences (for example Chinese-only vs bilingual, enabled/disabled)
  • Whether the first-run setup wizard has been completed

This data is not uploaded to servers operated by the Extension developer. Uninstalling the Extension removes this local storage according to Chrome’s normal behavior.

ZH 2. 儲存在你裝置上的資料

本擴充功能可能僅在你電腦的 Chrome 本機儲存（chrome.storage.local）中保存：

  • 你選擇的翻譯服務與模型名稱
  • API endpoint 網址
  • 你在設定精靈中貼上的 API key
  • 顯示偏好（例如只顯示繁中 / 雙語、是否啟用）
  • 是否已完成首次設定

上述資料不會上傳到擴充功能作者營運的伺服器。解除安裝後，將依 Chrome 機制清除本機擴充功能資料。

EN 3. Data sent over the network

When the Extension is active on a YouTube page with captions available:

  • Caption / subtitle text may be sent from your browser to the translation provider you configured (for example OpenAI, OpenRouter, Groq, a local Ollama server, or another OpenAI-compatible endpoint you specify).
  • Requests use the API key you provided (typically as a Bearer token to that provider).
  • The Extension developer does not operate a proxy that receives your API key or caption content by default.

Your use of third-party translation providers is also subject to those providers’ own privacy policies and terms (for example OpenAI’s policies).

ZH 3. 透過網路傳送的資料

當本擴充功能在 YouTube 頁面運作且有可用字幕時：

  • 字幕文字可能會從你的瀏覽器直接傳送到你設定的翻譯服務商（例如 OpenAI、OpenRouter、Groq、本機 Ollama，或其他你指定的 OpenAI-compatible endpoint）。
  • 請求會使用你提供的 API key（通常以 Bearer token 傳給該服務商）。
  • 擴充功能作者預設不營運會接收你的 API key 或字幕內容的中繼伺服器。

你使用第三方翻譯服務時，同時受該服務商自己的隱私權政策與服務條款拘束（例如 OpenAI 的政策）。

EN 4. Data we do not collect

  • No developer-operated user accounts for this Extension
  • No analytics SDKs shipped for advertising or cross-site tracking
  • No sale of personal information
  • No intentional collection of payment card data

ZH 4. 我們不蒐集的資料

  • 本擴充功能不要求在作者端建立帳號
  • 未內建用於廣告或跨站追蹤的分析 SDK
  • 不出售個人資訊
  • 不蒐集支付卡資料

EN 5. Permissions

The Extension requests only what it needs for subtitle translation on YouTube:

  • storage — save your settings and API key locally
  • Host access to YouTube — read caption UI / show the overlay
  • Host access to translation APIs — call the provider you select (and optional custom HTTPS endpoints)

ZH 5. 權限說明

本擴充功能只申請字幕翻譯所需權限：

  • storage — 在本機儲存設定與 API key
  • YouTube 網站存取 — 讀取字幕介面 / 顯示翻譯疊層
  • 翻譯 API 主機存取 — 呼叫你選擇的翻譯服務（以及可選的自訂 HTTPS endpoint）

EN 6. Children

The Extension is not directed at children under 13, and we do not knowingly collect personal information from children.

ZH 6. 兒童

本擴充功能並非以 13 歲以下兒童為對象，我們不會在知情情況下蒐集兒童的個人資訊。

EN 7. Changes

We may update this policy from time to time. The “Last updated” date at the top will change when we do. Continued use of the Extension after changes means you accept the updated policy.

ZH 7. 政策變更

我們可能不時更新本政策。更新時會修改文首的「最後更新」日期。若你在變更後繼續使用本擴充功能，即表示接受更新後的政策。

EN 8. Contact

For privacy questions about this Extension, contact the publisher via the email address listed on the Chrome Web Store listing, or open an issue on the project repository if one is linked from the listing.

ZH 8. 聯絡方式

若對本擴充功能的隱私有疑問，請透過 Chrome 線上應用程式商店商品頁所載的聯絡信箱與發行者聯繫；若商品頁有連結專案儲存庫，亦可在該處開 issue。
