# Security Policy

## Supported versions

Security fixes are applied to the latest released version. Older builds are not
supported unless a maintainer states otherwise.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability.

Use GitHub's private vulnerability reporting:

https://github.com/haru3613/realtime-subtitle-translator/security/advisories/new

Include the affected version, reproduction steps, impact, and any suggested
mitigation. You should receive an acknowledgement within seven days. Please
allow time for a fix before disclosing the issue publicly.
